3. Manual Backend Start Instructions

If backend is not running:

  1. Open Command Prompt or PowerShell
  2. Navigate to your project: cd C:\Users\pavit\OneDrive\Desktop\learn_to_code\Backend
  3. Start Flask: python app.py
  4. Look for message: "Running on http://127.0.0.1:5000"
  5. Return here and click "Test Connection"

4. Common Issues & Solutions

  • Connection refused: Backend not running
  • Problem M041 not found: ID mismatch between frontend and backend
  • CORS errors: Frontend-backend communication blocked
  • Empty response: Problems not loading from coding_questions folder
